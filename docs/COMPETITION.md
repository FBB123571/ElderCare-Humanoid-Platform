# 竞赛 / 创新创业答辩要点

## 一句话创新

**「多模态风险融合 + MediaPipe 视觉感知 + 主动式照护 + 人形机器人可部署接口」的养老陪伴系统。**

## 建议演示流程（5 分钟）

1. **Web 控制台**（2min）：打开 `http://localhost:8765`，展示仪表盘
2. **情感陪伴**（1min）：输入「最近有点孤独」→ 机器人回复 + 指令标签
3. **摄像头视觉**（可选 1min）：打开摄像头 → 分析当前帧 → 骨架叠加
4. **跌倒紧急**（1min）：点「模拟跌倒」或「运行演示剧本」→ emergency + 紧急呼叫
5. **部署说明**（30s）：ROS2 `/care/robot_cmd` + 技术报告/评测数据
