【重要】本项目在 Linux 服务器上，不要在 Windows CMD 里运行 bash 命令！

✅ 推荐：Web 控制台（本机浏览器即可使用）
-----------------------------------------
1. 在 Cursor 底部 SSH 终端执行：

   cd /mnt/sdb1/leijh/EnergySnake1/robot/ElderCare-Humanoid-Platform
   bash scripts/run_web.sh

2. Cursor 菜单：查看 → 端口（Ports），找到 8765，点击「在浏览器中打开」
   或手动打开：http://localhost:8765

3. 在网页里可以：
   - 调节心率/血氧/姿态，模拟老人状态
   - 输入老人话语，看机器人回复
   - 点「模拟跌倒」或「运行演示剧本」
   - 查看风险评分、机器人指令
   - 视觉分析：推荐「上传照片分析」（老人上半身照），比单人摄像头自拍更容易出骨架

4. 首次使用 MediaPipe 骨架检测前，在 SSH 终端执行（需能访问外网）：
   bash scripts/download_mediapipe_models.sh
   然后重启 run_web.sh

Isaac G1 仿真（本机观看 + 自己录屏）
------------------------------------
说明：远程 SSH 无法在你 Windows 上直接弹 3D 窗口，可用下面两种方式。

【方式 A】本机已装 Isaac Lab + 显示器（推荐）
  1. 把 checkpoint 拷到本机（或在本机下载，见 scripts/download_g1_checkpoint.sh）
  2. 在本机终端:
     export ISAACLAB_PATH=/path/to/IsaacLab
     export CHECKPOINT=/path/to/g1_flat_checkpoint.pt
     bash scripts/play_g1_local_gui.sh
  3. 弹出 G1 仿真窗口后，用 Win+G 或 OBS 录屏

【方式 B】仍在服务器跑，本机浏览器/客户端看直播
  1. SSH 终端: bash scripts/play_g1_livestream.sh
  2. Cursor → 端口 → 转发 49100
  3. 本机安装 NVIDIA Isaac Sim WebRTC Streaming Client，连 127.0.0.1:49100
  4. 对客户端窗口用 OBS 录制
  若报 Permission denied（isaacsim/kit/cache）→ 见 docs/ISAAC_FIX_PERMISSIONS.md

G1 仿真录屏（必须先修权限，只需一次）:
  cd /mnt/sdb1/leijh/EnergySnake1/robot/ElderCare-Humanoid-Platform
  sudo bash scripts/fix_isaac_permissions.sh    # 输入密码
  bash scripts/setup_and_record_g1.sh           # 自动录屏
  输出: docs/assets/demo_isaac_g1_locomotion.mp4

说明
----
- Windows CMD 不能运行 bash，请用 Cursor 底部终端。
- Web 方式最适合 Remote SSH：界面在你 Windows 浏览器里，逻辑在服务器上跑。
